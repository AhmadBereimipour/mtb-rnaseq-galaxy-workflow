# Reference genome information

Organism: Mycobacterium tuberculosis H37Rv

Dataset: GSE126286

Reference assembly: ASM19595v2

Annotation source: NCBI RefSeq

Annotation format used in Galaxy: GFF/GTF

Alignment tool: HISAT2

Quantification tool: FeatureCounts

Important note:
The reference genome used for HISAT2 alignment and the GFF/GTF annotation file used for FeatureCounts should correspond to the same genome assembly.
